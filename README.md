# Swift Ex Logistics Engine

The primary objective of the Swift Ex project was to develop a fully autonomous engine that manages the end-to-end lifecycle of parcels while maintaining high efficiency and data persistence without the use of the Standard Template Library (STL).

## Features
* **Custom Data Structures:** High-efficiency management without STL.
* **Persistent Storage:** File-based data handling for parcel records.
* **Tracking & Routing:** Real-time logic for parcel lifecycle management.
* **Intelligent Sorting:** Automated module for logistics optimization.
* **Security:** Secure console-based login system.
